# Audio File Conventions

| Aspect                   | Official Funkin                                                      | Psych Engine                                                                          | Codename Engine                                                   |
| ------------------------ | -------------------------------------------------------------------- | ------------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| **Instrumental**         | `songs/<id>/Inst.ogg` (suffix via variation, e.g. `Inst-erect.ogg`)  | `songs/<Song>/Inst.ogg`                                                               | `songs/<song>/song/Inst.ogg`                                      |
| **Vocals**               | Split per-character: `Voices-<charId>.ogg` (e.g. `Voices-bf.ogg`); for non-default variations: `Voices-<charId>-<variation>.ogg` (e.g. `Voices-bf-erect.ogg`). Auto-resolves hyphenated IDs | Single `Voices.ogg` (or optionally split: `Voices-Player.ogg`, `Voices-Opponent.ogg`) | Single `Voices.ogg` or split per-strumline via `vocalsSuffix` (e.g. `Voices_Opponent.ogg`) |
| **Vocal mapping**        | `characters.playerVocals` and `characters.opponentVocals` arrays in metadata | N/A                                                                            | `vocalsSuffix` per strumline in chart                             |
| **Alt instrumentals**    | `characters.instrumental` suffix + `characters.altInstrumentals` list | N/A                                                                                   | N/A                                                               |
| **Audio offsets**         | `offsets` object in metadata: `instrumental`, `altInstrumentals`, `vocals`, `altVocals` (all in ms) | N/A                                                              | N/A                                                               |
| **Inst suffix (meta)**   | N/A                                                                  | N/A                                                                                   | `meta.instSuffix` ‚Äî appended before difficulty (e.g. `Inst<suffix>.ogg`) |
| **Vocals suffix (meta)** | N/A                                                                  | N/A                                                                                   | `meta.vocalsSuffix` ‚Äî for main vocals; strumline `vocalsSuffix` for per-strumline |
| **Per-difficulty audio** | Via variations system (separate variation metadata + audio files)     | Separate song folders per difficulty                                                  | Tries `Inst-<difficulty>.ogg` first, falls back to `Inst.ogg`     |
| **Format**               | OGG Vorbis                                                           | OGG Vorbis                                                                            | OGG Vorbis                                                        |
| **Song variations**      | Full variation system (`erect`, `pico`, etc.) with separate metadata and audio | Not natively supported                                                     | Variant system: charts in `charts/<variant>/`, metadata in `meta-<variant>.json` |
