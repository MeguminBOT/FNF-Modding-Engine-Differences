# Codename Engine — Global Scripts

Place these in `data/scripts/` — they run globally across all game states.

## Overview

| Detail | Info |
|--------|------|
| **Location** | `data/scripts/<name>.hx` |
| **Language** | HScript (function-based) |
| **Key callbacks** | All gameplay callbacks (`create`, `update`, `beatHit`, etc.) |
| **Related docs** | [Scripting System](../scripting-system.md) · [Codename Events](../reference/codename-events.md) |
