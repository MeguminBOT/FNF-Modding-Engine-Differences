# Psych Engine — Global Scripts

Place these in `scripts/` — they run on every song regardless of stage or chart. Both `.lua` and `.hx` are supported.

## Overview

| Detail | Info |
|--------|------|
| **Location** | `scripts/<name>.lua` or `.hx` |
| **Languages** | Lua, HScript (Iris) |
| **Key callbacks** | All gameplay callbacks (`onCreate`, `onUpdate`, `goodNoteHit`, etc.) |
| **Loaded when** | Always — runs on every song automatically |
| **Related docs** | [Script Callbacks](../script-callbacks.md) · [Scripting System](../scripting-system.md) · [Psych Lua API](../reference/psych-lua-api.md) |
