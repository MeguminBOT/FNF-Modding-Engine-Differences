# Psych Engine — Event Scripts

Place these in `custom_events/` — the filename (without extension) becomes the event name in the chart editor. Both `.lua` and `.hx` are supported.

## Overview

| Detail | Info |
|--------|------|
| **Location** | `custom_events/<Name>.lua` or `.hx` |
| **Languages** | Lua, HScript (Iris) |
| **Key callbacks** | `onEvent(name, value1, value2)` |
| **Related docs** | [Events System](../events-system.md) · [Scripting System](../scripting-system.md) · [Psych Lua API](../reference/psych-lua-api.md) |
