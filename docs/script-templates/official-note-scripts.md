# Official Funkin' — Note Scripts

Custom note types are handled through HScript module scripts (`.hxc`).

## Overview

| Detail | Info |
|--------|------|
| **Location** | `scripts/<name>.hxc` |
| **Language** | HScript (class-based, `.hxc`) |
| **Key pattern** | `class extends Module` with note handler overrides |
| **Related docs** | [Note Types](../note-types.md) · [Scripting System](../scripting-system.md) |
