# Official Funkin' — Global Scripts

Module scripts in `scripts/` run globally (`.hxc`).

## Overview

| Detail | Info |
|--------|------|
| **Location** | `scripts/<name>.hxc` |
| **Language** | HScript (class-based, `.hxc`) |
| **Key pattern** | `class extends Module` with lifecycle overrides |
