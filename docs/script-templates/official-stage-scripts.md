# Official Funkin' — Stage Scripts

Stages are JSON-defined with optional scripted behavior via module scripts (`.hxc`).

## Overview

| Detail | Info |
|--------|------|
| **Location** | `scripts/stages/<name>.hxc` |
| **Language** | HScript (class-based, `.hxc`) |
| **Key pattern** | `class extends Stage` with `create`, `beatHit` overrides |
| **Related docs** | [Stage Format](../stage-format.md) · [Scripting System](../scripting-system.md) |
