# Codename Engine — Note Scripts

Place these in `data/notes/` — the filename (without extension) becomes the note type name. Codename can also auto-replace note sprites from `images/game/notes/<type>/`.

## Overview

| Detail | Info |
|--------|------|
| **Location** | `data/notes/<Name>.hx` |
| **Language** | HScript (function-based) |
| **Key callbacks** | `onNoteCreation(event)`, `onPlayerHit(event)`, `onPlayerMiss(event)`, `onNoteHit(event)` |
| **Related docs** | [Note Types](../note-types.md) · [Scripting System](../scripting-system.md) · [Codename Events](../reference/codename-events.md) |
