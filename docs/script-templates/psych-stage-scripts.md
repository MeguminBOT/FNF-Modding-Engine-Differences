# Psych Engine — Stage Scripts

Loaded from `stages/<name>.lua` or `.hx` alongside the stage JSON. Both `.lua` and `.hx` are supported.

## Overview

| Detail | Info |
|--------|------|
| **Location** | `stages/<name>.lua` or `.hx` |
| **Languages** | Lua, HScript (Iris) |
| **Key callbacks** | `onCreate`, `onCreatePost`, `onUpdate`, `onBeatHit`, `onStepHit` |
| **Related docs** | [Stage Format](../stage-format.md) · [Scripting System](../scripting-system.md) · [Psych Lua API](../reference/psych-lua-api.md) |
